# 💰 Expense Tracker

A simple and responsive **Expense Tracker** built using **HTML, CSS, and JavaScript**. This application helps users manage their daily income and expenses by tracking transactions, calculating balances, and storing data locally in the browser.

---

## 🚀 Features

- ➕ Add Income and Expense Transactions
- 🗑️ Delete Transactions
- 💾 Data Persistence using Local Storage
- 📊 Automatic Balance Calculation
- 💵 Total Income and Total Expense Summary
- 🌙 Dark Mode Toggle
- 📤 Export Transactions to CSV
- 📱 Responsive Design for Mobile and Desktop
- 🎨 Modern User Interface

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/ShivamKumarSrivastava/Expense-Tracker.git
```

2. Open the project folder

```bash
cd Expense-Tracker
```

3. Open `index.html` in your browser.

No additional setup is required.

---

## 📖 How to Use

1. Enter the transaction title.
2. Enter the amount.
3. Select **Income** or **Expense**.
4. Choose a category.
5. Select a date.
6. Click **Add Transaction**.
7. View your updated balance and transaction history.
8. Delete transactions whenever needed.
9. Export your data as a CSV file.

---

## 💾 Data Storage

All transaction data is stored in the browser using **Local Storage**, so your data remains available even after refreshing the page.
